メモ帳詰め合わせ

Release.zipを展開して、実行ファイルを実行するだけです。
2種類の簡易メモ帳を同梱しています。
インターネットからのダウンロードのため、実行できない場合は、ブロックを解除してください。


************************
*  メモ帳V1 (100行くらい)
*  テンプレ対応日記帳
*  NMemoPad.exe
************************
  - 原稿を書き終えたら、保存します。
  - NMemoPad.settingsを変更することでテンプレートを変更できます。
